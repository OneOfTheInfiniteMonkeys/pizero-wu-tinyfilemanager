# Place holder for temporary files created by pyembroidery extension to 
# PiZero-WU Tiny File Manager
#
# When an embroidery file is rendered it is stored in this folder to permit
# storage and display in the users web browser.
# Only 1 file 'ebi.svg' will be stored in this folder at a time.
# Each time a new file is viewed the 'ebi.svg' will be overwritten.
#
# The default ebi.svg rendering file in this folder is a rendering of Tux
# from the included file Pinguin23.pes
# Source for this file was:
# http://free-penguin.org/
# Tux (Penguin) design is copyright Larry Ewing
# You might find this interesting (hence Pinguin23.pes):

# 
# Following installation this text file may be removed if desired.
